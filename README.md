## :package: 安装

- 安装 Stun

  进入 Hexo 根目录，执行指令：

  ```bash
  $ git clone https://github.com/liuyib/hexo-theme-stun.git themes/stun
  ```

- 安装依赖 `hexo-renderer-pug`

  进入 Hexo 根目录，执行指令：

  ```bash
  $ npm install --save hexo-renderer-pug
  ```

## :hammer: 使用

修改 Hexo 根目录下的 `_config.yml` 文件：

```yml
theme: stun
```

然后，启动 Hexo 服务器：

```bash
$ hexo clean && hexo s
```
