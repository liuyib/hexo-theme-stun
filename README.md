## :package: Install

Enter your hexo directory, run this:

```bash
$ git clone https://github.com/liuyib/hexo-theme-stun.git themes/stun
```

## :hammer: Usage

Change the `_config.yml` file in your hexo root directory:

```yml
theme: stun
```

Run your hexo server:

```bash
$ hexo clean && hexo s
```

## :art: Update

```bash
$ cd themes/stun

$ git pull
```
